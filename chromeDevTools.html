Chrome Dev Tools

  1. Use "debugger" in script as breakpoint
  2. $('a') === document.querySelector('a')  returns first element match
  3. $$('a') === document.querySelectorAll('a')  returns all matching elements
  4. Right click and perserve log upon navigation  reload/refresh will persist logs
  5. Right click on element and select Break on (subtree modified, attributes modified, node removal)  helps find out which JS is called on interaction with element
  6. document.body.contentEditable = true  your HTML screen will be editable
  7. console.log('Window', window, 'Body', document.body)  will log window and body objects along with labels
  8. $('#myInput')  will log HTML output of element
  9. dir($('#myInput'))  will log all properties of element
  10. getEventListeners($('#myInput'))  will log event listeners for element
  11. monitorEvents($('#myInput'))  will log all events attached to element when you interact with it
  12. monitorEvents($('#myInput'), ['click', 'blur'])  will log click and blur events attached to element
  13. unmonitorEvents($('#myInput'))  will unmonitor events
  14. console.time('myTime') and console.timeEnd('myTime')  capture elapsed time, use in code to benchmark functions etc.
  15. console.groupCollapsed('My Errors') and console.groupEnd()  collapse/group your logs
  16. console.table(myArray)  log a table view of array elements
  17. $_  access and use result of last expression
  18. clear() or ctrl + L  clear the console log
  19. console.trace()  get call stack at anytime
  20. console.count('fooed')  logs count everytime called
  21. profile('myProfile') and profileEnd('myProfile')  go to Profiles tab to see results
  22. inspect($('#myInput'))  will highlight in Elements tab and in browser view
  23. $0  logs last selected element, $1 will log 2nd to last element etc. You can alternatively add ID to element and get it by ID
  24. ctrl + [minus] and ctrl + [plus]  will zoom in/out browser view while in console
  25. arrow up and arrow down  displays last console commands
  26. copy(document.body.innerText)  copy body text to clipboard
  27. edit JS directly in Scripts tab  add logs to help debug perhaps
  28. click on pause button on lower left to debug exception  this will take to directly to JS exception
  29. drag and drop elements  do this in Elements tab to shift elements and see display
  30. In settings, there's a checkbox to Log XMLHttpRequests  
  31. https://ericduran.github.io/chromeHAR/  save and drop your HAR results here
